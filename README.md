# AntiCivBreak

AntiCivBreak is a Minecraft Paper/Spigot plugin designed to prevent destructive block breaking and log suspicious player activity.

## Features

- Civ Break protection system
- Suspicious block break logging
- Log GUI viewer
- Multi-language support
- Lightweight and optimized

## Commands

/acb reload
Reload the plugin configuration.

/acb log
Open the block break log GUI.

## Permissions

anticivbreak.admin

## Supported Versions

Paper / Spigot 1.21+

## Installation

1. Download the latest jar from Releases
2. Place it in your server `plugins` folder
3. Restart the server

## Configuration

config.yml

You can configure:

- protected blocks
- logging settings
- language
